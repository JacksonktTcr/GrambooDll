## Timeline

```
User Action: dgv.SummaryColumns = new string[] { "Amount" }
?
?? T0: Property setter called
?? T1: Conditions checked (not null, has values, not paused)
?? T2: SummaryRowVisible = true (calls property setter)
?? T3: RefreshSummary(true) called automatically
?? T4: Summary boxes created in UI
?? T5: Calculations performed on grid data
?? T6: Values formatted and stored in boxes
?? T7: Invalidate() called for repaint
?? T8: Paint event triggered
?? T9: Summary row visible on screen
?
?? ? Complete! (All automatic, <100ms typically)
```

## Summary

```
?? GOAL: Simplify summary row setup
? FEATURE: Auto-show and calculate when columns assigned
?? IMPLEMENTATION: Enhanced SummaryColumns property setter
?? PROCESS: Property ? Validation ? Visibility ? Refresh ? Display
?? RESULT: One-line setup instead of three!
```

---

**Key Concept:** Setting `SummaryColumns` triggers an automatic chain of events that creates and displays the summary row without requiring additional configuration steps.

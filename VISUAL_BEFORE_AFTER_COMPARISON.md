## One Line That Matters

### The Critical Change
```csharp
// BEFORE:
e.Graphics.DrawString(Text, Font, Brushes.Black, textBounds, format);
// Problem: Text might be null or empty

// AFTER:
e.Graphics.DrawString(displayText, Font, Brushes.Black, textBounds, format);
// Solution: displayText is guaranteed to have a value
```

## Implementation Pattern

### Safe Display Pattern Used
```
1. START: Provide safe default
   string display = value ?? "0"

2. ATTEMPT: Improve the value
   try { display = format(value); }

3. FALLBACK: Return to safe value
   catch { display = value ?? "0"; }

4. USE: The guaranteed-safe value
   return display;
```

This pattern ensures **blank displays never happen again**!
